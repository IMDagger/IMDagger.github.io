Unix PID-файл

Дополнительная библиотека, которая легко ставится (даже через старый добрый easy_install) lockfile позволяет делать блокировки на файлы и запросто проверять блокировку с учётом таймаута.

[imdagger@killer ~]$ sudo easy_install lockfile
хохохо! тут скачивается яйцо
…
запускаем python
import lockfile
from lockfile import pidlockfile
f = pidlockfile.PIDLockFile('/var/run/mydaemon.pid')
try:
    f.acquire(1) # timeout в 1 секунду, попытка залочить файл
except lockfile.LockTimeout:
    print 'Already running pid=%s...' % f.read_pid() # уже есть рабочая копия демона

В пакете присутствуют ещё удобные моудли для директорий, симлинков и т.д. Вкусная и простая библиотека, к тому же кроссплатформенная. И не тянет за собой ничего лишнего, чистый Python, стандартные библиотеки. Класс PIDLockFile создаёт pid файл в соответствии с требованиями спецификации HFS 2.3.
